# Desafio Técnico - Soluções

Este repositório contém as soluções para o desafio técnico proposto. O objetivo do desafio foi resolver uma série de problemas lógicos e computacionais. Abaixo, você encontrará uma breve descrição de cada questão e como rodar os scripts.

## Questões

### 1. **Soma com Loop**
- Objetivo: Soma os números de 1 até o valor de `INDICE` utilizando um loop.
- Arquivo: [questao1_soma_loop.js](questao1_soma_loop.js)

### 2. **Verificação de Número na Sequência de Fibonacci**
- Objetivo: Verificar se um número informado pertence à sequência de Fibonacci.
- Arquivo: [questao2_fibonacci.js](questao2_fibonacci.js)

### 3. **Análise de Faturamento Diário**
- Objetivo: Calcular o menor valor de faturamento, o maior valor e os dias com faturamento superior à média.
- Arquivo: [questao3_faturamento_diario.js](questao3_faturamento_diario.js)

### 4. **Cálculo do Percentual de Faturamento por Estado**
- Objetivo: Calcular o percentual de faturamento de cada estado dentro do faturamento total.
- Arquivo: [questao4_percentual_estado.js](questao4_percentual_estado.js)

### 5. **Inversão de String**
- Objetivo: Inverter os caracteres de uma string sem usar funções prontas como `reverse()`.
- Arquivo: [questao5_inverter_string.js](questao5_inverter_string.js)
